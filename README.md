# 📚 Book Tracker

Трекер прочитанных книг — GUI-приложение для управления списком книг с фильтрацией и статистикой.

---

## 📋 Описание

Приложение позволяет:
- ✅ Добавлять книги с указанием названия, автора, жанра и количества страниц
- ✅ Просматривать все книги в виде таблицы
- ✅ Фильтровать книги по жанру и количеству страниц
- ✅ Подсчитывать общее количество прочитанных страниц
- ✅ Экспортировать отчёт в текстовый файл
- ✅ Автоматически сохранять данные в JSON-файл

## 📦 Требования

### Версия Python
- **Python 3.6** или выше
- Рекомендуется Python 3.8+

### Зависимости
Все используемые модули входят в **стандартную библиотеку Python**:

| Модуль | Описание |
|--------|----------|
| `tkinter` | Графический интерфейс (встроен) |
| `json` | Работа с JSON файлами (встроен) |
| `os` | Работа с файловой системой (встроен) |
| `unittest` | Тестирование (встроен) |

> ⚠️ **Примечание:** На некоторых Linux-системах tkinter может требовать отдельной установки:
> ```bash
> # Ubuntu/Debian
> sudo apt-get install python3-tk
> 
> # Fedora
> sudo dnf install python3-tkinter
> 
> # macOS
> brew install python-tk
> ```
